Programa para gestionar el inventario de una red de almacenes.

Una vez al año debe hacerse un inventario valorizado del stock de piezas de ordenador que hay en los almacenes de una fabrica. Se sabe que la industria cuenta con M almacenes y N piezas.
El programa implementa una matriz inventario donde se almacena el stock, donde ( M representa los almacenes) y ( N representa las piezas).
Se usan también, un vector donde se almacenan los nombres de las piezas y un vector donde se almacenan los precios de cada una de las piezas.

El programa:
- Pide y registra el número de almacenes, la cantidad de piezas, los nombres de las piezas y sus precios,
- indica, dado un almacén indicado por el usuario, el valor total de cada tipo de pieza almacenada,
- indica el valor total de cada almacén,
- indica el total sumado del valor de todas las piezas en todos los almacenes, e
- indicar en cuales almacenes no hay stock de de alguna de las piezas.
